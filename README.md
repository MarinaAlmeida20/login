# login Page

This is Login Page created to learning how to create a Design System.

## Technologies

- Vite
- React
- TypeScript
- Tailwind
- Storybook
- Radix UI
- phosphor-react
- Axios
- MSW

## Installation

Use the package manager npm to install all dependencies.

```bash
npm install
npm run dev
```
